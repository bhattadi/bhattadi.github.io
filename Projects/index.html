Weather Effect of Music Analyzer

04/27/2020

This Python script calls the Spotify API, MetaWeather API, and Billboards API to gather data as JSON objects and store them in a sqlite3 database. I collected data from 2013 to 2020 and produced visualizations using Matplotlib to analyze what genre of music is most popular during different weather conditions.
